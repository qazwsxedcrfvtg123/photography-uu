React + Vite
這個模板提供了一個最精簡的設置，讓 React 可以在 Vite 中運行，並支援 HMR（熱模組替換，修改程式碼網頁能不重整即時更新）以及一些基礎的 ESLint 規則。
Vite是一個現代的前端建置工具 (Build Tool) 與開發伺服器。
目前提供兩個官方外掛：

@vitejs/plugin-react：使用 Babel（或在使用 rolldown-vite 時使用 oxc）來實現快速刷新 (Fast Refresh)。
@vitejs/plugin-react-swc：使用 SWC 來實現快速刷新。
React Compiler (React 編譯器)
這個模板預設沒有啟用 React Compiler，因為它對開發與建置的效能會造成影響。
